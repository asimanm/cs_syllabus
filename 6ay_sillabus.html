6-Month CS & Data Science Syllabus

1st year university student · Germany · Java → C → Python + Data Structures → Algorithms → Data Science → Deep Learning & AI

Duration 6 months / 24 weeks
Sessions 3 × 90 min / week
Total ~108 hours
Prerequisite loops, if/else, functions, arrays

Month 1 — Java: OOP Fundamentals

One session of Java syntax setup, then straight into OOP. Classes, encapsulation, inheritance, interfaces, generics, exceptions. 3 sessions × 90 min/week.

Month 2 — C: Memory, Pointers & Systems

C shows what Java hides. Pointers, malloc/free, structs, file I/O, linked list in C, debugging with gdb and Valgrind.

Month 3 — Python + Data Structures

2 weeks on Python from a Java/C background, then 2 weeks implementing every major data structure — both theory and working Python code.

Month 4 — Algorithms & CS Theory

Big-O, sorting, searching, graph algorithms, and dynamic programming — theory first, then implemented in Python.

Month 5 — Data Science: NumPy, Pandas & ML

NumPy, Pandas, EDA, visualization, statistics foundations, classical ML with scikit-learn pipelines.

Month 6 — Deep Learning & Modern AI

Neural networks, PyTorch, CNNs, transformers, LLMs, fine-tuning, and a final capstone project to show employers.
